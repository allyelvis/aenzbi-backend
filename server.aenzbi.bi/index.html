AENZBi Backend API

This API powers the AENZBi Business Management and POS system.

Endpoints

Authentication

POST /auth/login

Authenticate users and receive a JWT token.

Sales

POST /sales

Record sales transactions. Requires JWT.

Products

GET /products

Retrieve a list of available products.

User Info

GET /users/me

Fetch user details (protected). Requires JWT.
