AENZBi Backend API

Welcome to the AENZBi Business Management and POS API. Use the endpoints below to manage transactions, inventory, users, and more.

Endpoints

  • Authentication

    POST /auth/login - Authenticate and receive a JWT token.

  • Sales

    POST /sales - Record sales transactions.

  • Products

    GET /products - Fetch list of available products.

  • Users

    GET /users - Manage and fetch user details.

  • Inventory

    POST /stock - Record stock movements.

  • Reports

    GET /reports - Generate system reports.
